Preguntas Verdadero-Falso

Teniendo encuenta los ejemplos de sistemas operativos de escritorio, responde las siguientes preguntas:

Pregunta 1

¿Microsoft Windows desarrollo el Sistema Operativo Ubuntu?

Retroalimentación

Falso

Ubuntu es una distribución de Linux

Pregunta 2

¿Mac OS X es un sistema operativo desarrollado exclusivamente por Apple?

Retroalimentación

Verdadero

Apple se ha caracterizado por desarrollar sistemas operativos únicos y diferentes, enfocados en el trabajo multimedia.

Pregunta 3

¿Debían es una versión de sistema operativo desarrollado por Microsoft Windows?

Retroalimentación

Falso

Debian es una distribución de Linux

Pregunta 4

¿Los sistemas operativos de Microsoft Windows, como windows 7 son de libre distribución o acceso?

Retroalimentación

Falso

Todos los sistemas operativos de Microsoft Windows lanzados hasta la fecha son de Licencia Pagada.

Pregunta 5

¿Los sistemas operativos basados en Linux pueden ser módificados por cualquier usuario?

Retroalimentación

Verdadero

Las distribuciones de Linux son de código abierto por lo tanto pueden ser modificados por cualquier usuario
